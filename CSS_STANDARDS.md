# CSS Design System Standards

## Color Palette
- Primary: #3b82f6 (Blue)
- Primary Dark: #2563eb
- Secondary: #6b7280 (Gray)
- Background: #f8fafc (Light Gray)
- White: #ffffff
- Text Dark: #111827
- Text Light: #6b7280
- Border: #e5e7eb

## Spacing (Standard: 4px units)
- xs: 4px
- sm: 8px
- md: 12px
- lg: 16px
- xl: 20px
- 2xl: 24px
- 3xl: 28px

## Border Radius
- Small: 8px (inputs, buttons)
- Medium: 10px (cards, boxes)
- Large: 12px (modals, large containers)
- Circle: 50% (avatars, badges)

## Box Shadow
- sm: 0 1px 2px rgba(0, 0, 0, 0.03)
- md: 0 2px 8px rgba(0, 0, 0, 0.04)
- lg: 0 4px 12px rgba(0, 0, 0, 0.05)
- xl: 0 8px 16px rgba(0, 0, 0, 0.08)

## Typography
- Body font-family: "Inter", sans-serif
- Headings: 700 weight
- Labels: 600 weight
- Body text: 400 weight
- Small text: 13px

## Components Standardized
✓ Buttons (all variants)
✓ Cards
✓ Boxes
✓ Form inputs
✓ Tables
✓ Top bars
✓ Colors and themes
✓ Spacing and gaps
✓ Shadows
